Release History

A summary of OMERO.figure releases. Follow the links for full release announcements:

1.2.1: {{ "2016-04-20" | date_to_string }}

  • Improvements to performance and filtering in the File > Open dialog
  • Small number of minor bug fixes

1.2.0: {{ "2015-11-24" | date_to_string }}

  • Addition of ROIs on figure panels
  • Draw rectangles, ellipses, lines and arrows
  • Copy and paste ROIs between image panels
  • Use ROI rectangles as crop regions on other images

1.1.1: {{ "2015-05-05" | date_to_string }}

  • Minor release with a small number of bug-fixes and tweaks

1.1.0: {{ "2015-04-20" | date_to_string }}

  • Multi-page figures
  • Crop panel images to region or ROI
  • Add figure legend
  • Export of figures as TIFF images

1.0.1: {{ "2015-04-02" | date_to_string }}

  • Minor release to add compatibility for OMERO 5.1

1.0.0: {{ "2014-10-07" | date_to_string }}

  • First non-beta release
  • Maximum-intensity projection
  • Z and T index incrementing

1.0.0-beta2: {{ "2014-06-21" | date_to_string }}

  • Various bug fixes in the web UI and figure export script
  • File > Open file list sortable and filtered by owner

1.0.0-beta1: {{ "2014-03-11" | date_to_string }}

  • First beta release
